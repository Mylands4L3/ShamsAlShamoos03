خوبه — مرحله‌به‌مرحله و کاملاً امن جلو می‌ریم. من چند روش ساده و مطمئن با PowerShell می‌دم (هم برای پوشه‌های سطح اول، هم برای بازگشتی — یعنی همه زیرپوشه‌ها). اول «پیش‌نمایش» انجام می‌دیم تا ببینی چه چیزهایی تغییر خواهند کرد، بعد که مطمئن شدی دستور واقعی رو اجرا می‌کنی.

آماده‌سازی

به پوشه‌ی اصلی برو (پوشه‌ای که می‌خواهی داخلش فایل و پوشه‌ها را تغییر نام بدهی).

داخل همان پوشه Shift + راست‌کلیک → Open PowerShell window here (یا از File Explorer آدرس را کپی کن و در PowerShell با cd "C:\مسیر\پوشه" برو).۱) فقط «پیش‌نمایش» — چه چیزهایی قرار است تغییر کنند

(این دستور فقط نمایش می‌دهد؛ هیچ چیز را تغییر نمی‌دهد)

Get-ChildItem -Recurse -Force |
Where-Object { $_.Name -match "(?i)GRD" } |
ForEach-Object {
    $newName = $_.Name -replace '(?i)GRD', 'Arrest'
    Rename-Item -Path $_.FullName -NewName $newName
}
